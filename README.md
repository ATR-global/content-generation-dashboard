# Starter Project

Starter scaffold based on the `sparked-ui` directory conventions.

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run preview` - preview production build
- `npm run type-check` - run Vue + TypeScript checking
- `npm run build:widget` - build web component bundle

## Directory Structure

- `src/assets` - static source assets and styles
- `src/cdns` - custom element/web component entrypoints
- `src/components` - reusable UI components
- `src/composables` - reusable Vue composition logic
- `src/locales` - translation dictionaries
- `src/router` - route definitions and navigation hooks
- `src/services` - API/service layer
- `src/stores` - Pinia stores
- `src/types` - shared TypeScript types
- `src/utils` - utility helpers
- `src/views` - route-level pages
- `public` - public assets served as-is
- `scripts` - project scripts
